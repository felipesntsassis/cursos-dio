Surgimento da Internet

Criada pelo britânico Tim Berners-Lee, o acrônimo HTML significa Hyper Text Markup Language, traduzindo para o português: Linguagem de Marcação de Hipertexto. O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página web. Sem isso, o navegador não saberia exibir textos, elementos ou carregar imagens e outros conteúdos.

Os hipertextos são conjuntos de elementos conectados. Esses podem ser palavras, imagens,vídeo, documento, etc. Quando conectados, formam uma rede de informações que permite a comunicação de dados, organizando conhecimentos e guardando informações.

Ao visitar uma página simples na web, você pode perceber que existem diferentes distribuições e tamanhos para títulos, parágrafos, imagens, vídeos e qualquer outro elemento. Essa é a estrutura estabelecida através do HTML. No início da web, era comum encontrar apenas textos e imagens simples, com estrutura básica e sem estilizações. Porém nos dias atuais, muito dificilmente você encontrará sites que possam conter apenas elementos HTML. Portanto, podemos considerar o HTML como o "esqueleto" da sua página.

Imagine então que além do esqueleto, é necessário ter corpo. Para isso temos então as linguagens CSS e o JavaScript, que em conjunto com o HTML formam a base das páginas atuais. Veremos mais a frente o que significam essas linguagens.

Como funciona o HTML

(Voltar)

Através de um documento HTML, ou seja, um documento com a extensão .html, ou .htm, o navegador faz a leitura do arquivo e renderiza o seu conteúdo para que o usuário final possa visualizá-lo. Os arquivos HTML podem ser visualizados em qualquer navegador (Google Chrome, Firefox, Edge, Safari, ou Opera).

O código pode ser escrito de qualquer editor de texto, como o próprio Bloco de Notas. Cada página consiste em uma série de tags (também chamados de elementos) que podem ser considerados os blocos de construção das páginas. Portanto, esses blocos são a maneira com a qual o HTML faz a marcação dos conteúdos, criando a hierarquia e a estrutura do mesmo, dividindo entre seções, parágrafos, cabeçalhos e outros.

O que são Tags

(Voltar)

Utilizando um editor de textos, como Sublime Text, o Notepad ++, ou até mesmo o Bloco de Notas, é possível criar um documento com extensão .html que será renderizado pelos navegadores.

Conforme explicado anteriormente, este documento consite em uma série de tags. As tags são códigos que definem toda a estrutura da página, tais como o seu tamanho, as quebras de linha e etc. A maioria dos elementos do documento HTML são compostos por uma estrutura de abertura e uma de fechamento, como é. Há também tag que realiza quebra de linha.

Digamos que você queria escrever um parágrafo, chamandos então a tag "p", escrevemos o parágrafo e finalmente fechamos a tag com "/p".

"Meu primeiro parágrafo."

Dessa forma, ao salvar o arquivo com extensão .html e abrí-lo em um navegador, você verá um parágrafo escrito na tela do navegador.
